# Thread 그룹
* 서로 관련된 쓰레드를 그룹으로 묶어서 다루기 위한 것
* 모든 쓰레드는 반드시 하나의 쓰레드 그룹에 포함되어야한다.
* 쓰레드 그룹을 지정하지 않고 생성한 쓰레드는 main쓰레드 그룹에 속한다.

```java
ThreadGroup getThreadGroup() // 쓰레드 자시이 속한 쓰레드 그룹을 반환한다.
```
